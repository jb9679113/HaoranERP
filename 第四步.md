实现 Dashboard.jsx 综合仪表盘页面：

数据从 Supabase 查询，今日/本月数据用 JavaScript 在前端按日期过滤。

布局：
1. 顶部「进销概览」标题 + 4张 StatCard：
   - 今日销售额（蓝色图标 TrendingUp）
   - 本月销售额（绿色图标 DollarSign）
   - 本月采购额（橙色图标 ShoppingCart）
   - 本月毛利润 = 销售额 - 采购成本（青色图标 Activity）

2. 「经营流水」标题 + 3张 StatCard：
   - 本月入账合计（绿色）
   - 本月支出合计（红色）
   - 本月净额（蓝色）

3. 左侧：过去30天销售额折线图（Recharts LineChart，X轴日期 MM-DD，Y轴金额 ¥）
4. 右侧：低库存预警卡片（stock_quantity ≤ 10 的商品，数量红色加粗）
5. 底部：最近10条销售记录表格（商品名、客户、销售员、数量、金额、日期）

StatCard 组件接收 { title, value, icon, iconBgColor } props，
value 用 lib/format.js 的 formatCurrency 格式化为 ¥#,##0.00
